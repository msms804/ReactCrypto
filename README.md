## 4. 주요 기능
* 실시간 가격 제공
* 코인테마별 분류, 테마별 정보 (gcTime, staleTime 조정하여 30분에 한번씩 갱신되도록 설정)
* 코인 검색
* 코인 차트
* 이름, 가격, 거래대금 별 정렬
* 관심코인 watchlist (로컬스토리지 저장)
* 다크모드 (로컬스토리지 저장)
